서울에서 경산까지 (tiny)

CodeUp 4880

풀러 가기

힌트 0 (시간/공간 복잡도)

시간 복잡도는 \( O(2^{N}) \)입니다.
공간 복잡도는 \( O(N) \)입니다.

힌트 1

\( N \)이 많이 작네요. 그냥 모든 경우를 시도해볼 정도로요.

풀이

가능한 모든 경우의 수는 \( 2^{N} = 2^{10} = 1024 \)가지이므로,
그냥 모든 경우를 백트래킹 등으로 돌려보면 됩니다.

코드 (C++)

#include <bits/stdc++.h>
#define endl '\n'
using ll = long long;
using namespace std;

class place{ public:
    int wt, wm;
    int ct, cm;
};

int n, k;
place arr[20];
int ans = 0;

void btk(int idx, int tim, int mon){
    if (idx > n){
        if (tim <= k){ ans = max(ans, mon); }
        return;
    }
    place p = arr[idx];
    btk(idx+1, tim + p.wt, mon + p.wm);
    btk(idx+1, tim + p.ct, mon + p.cm);
}

void solve(){
    cin >> n >> k;
    for (int i = 1; i <= n; i++){
        cin >> arr[i].wt >> arr[i].wm
            >> arr[i].ct >> arr[i].cm;
    }
    btk(1, 0, 0);
    cout << ans;
}

int main(){
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    solve();
}
