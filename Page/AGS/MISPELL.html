힌트 0 (시간/공간 복잡도)

시간 복잡도는 \( O(|S|) \)입니다.
공간 복잡도는 \( O(|S|) \)입니다.

힌트 1

주어진 문자열에서 한 글자를 빼는 문제입니다.

풀이

주어진 문자열을 그대로 한 글자 한 글자 출력하되
특정 위치의 문자만 빼고 출력해주면 됩니다.

코드 (C++)

#include <bits/stdc++.h>
#define endl '\n'
using namespace std;

void solve(){
    int t; cin >> t;
    for (int tt = 1; tt <= t; tt++){
        int idx; cin >> idx; idx -= 1;
        string s; cin >> s;
        int sl = s.size();
        cout << tt << ' ';
        for (int i = 0; i < sl; i++){
            if (idx == i){ continue; }
            cout << s[i];
        }
        cout << endl;
    }
}

int main(){
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    solve();
}
