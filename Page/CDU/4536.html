풀러 가기

힌트 0 (시간/공간 복잡도)

시간 복잡도는 \( O(N) \)입니다.
공간 복잡도는 \( O(N) \)입니다.
여기서 \( N \)은 입력값의 범위 (1000)입니다.

힌트 1

이 문제에서 평균은 모든 수를 합한 뒤 10으로 나눠주면 됩니다.
그럼 최빈값은 어떻게 구할까요?

힌트 2

최빈값은 가장 자주 나온 수입니다.
그렇다면 최빈값을 구하기 위해서는 각 수가 나온 횟수를 저장하면 됩니다.

풀이

10개의 수를 입력받으면서, 입력받은 수들의 합을 저장하면 됩니다.
그와 동시에, 각 수의 등장 횟수를 저장하는 배열을 만든 뒤, 그 배열에 등장 횟수를 저장해주면 됩니다.

입력받은 수들의 합을 저장했으므로, 평균값은 합 / 10으로 구할 수 있고,
각 수의 등장 횟수를 저장했으므로, 최빈값(가장 자주 등장한 수)을 찾을 수 있습니다.

코드 (C++)

#include <bits/stdc++.h>
#define FastIO ios_base::sync_with_stdio(0); cin.tie(0); cout.tie(0);
#define endl '\n'
using namespace std;

int cnt[1020]; // 등장 횟수 저장

void solve(){
    int sum = 0; // 합 저장
    for (int i = 1; i <= 10; i++){
        int x;
        cin >> x;
        sum += x; // 합 업데이트
        cnt[x] += 1; // 등장 횟수 업데이트
    }
    int avg = sum / 10, mode = 0, mxc = 0; // 평균, 최빈값, 최대 등장 횟수
    for (int i = 1; i <= 1000; i++){ // 최빈값 찾기
        if (mxc < cnt[i]){
            mode = i;
            mxc = cnt[i];
        }
    }
    cout << avg << endl << mode;
}

int main(){
    FastIO;
    solve();
}
