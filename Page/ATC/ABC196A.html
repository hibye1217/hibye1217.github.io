풀러 가기

번역

정수 \( a, b, c, d \)가 주어집니다.
이제, \( a \lt x \lt b \)이고 \( c \lt y \lt d \)인 정수 \( x, y \)를 고릅니다.
가장 큰 \( x - y \)를 찾으세요.

힌트 0 (시간/공간 복잡도)

시간 복잡도는 \( O(1) \)입니다.
공간 복잡도는 \( O(1) \)입니다.

힌트 1

\( x - y \)가 커지려면 \( x \)와 \( y \)는 무슨 값으로 잡는 것이 좋을까요?

풀이

\( x - y \)를 최대화하려면,
\( x \)를 가능한 크게 잡고, \( y \)를 가능한 작게 잡아야 합니다.

\( x \)의 최댓값은 \( b \)이고, \( y \)의 최솟값은 \( c \)이므로
\( b - c \)를 출력해주면 됩니다.

코드 (C++)

#include <bits/stdc++.h>
#define endl '\n'
using ll = long long;
using namespace std;

void solve(){
    int a, b, c, d;
    cin >> a >> b >> c >> d;
    cout << b - c;
}

int main(){
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    solve();
}
