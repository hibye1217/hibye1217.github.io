무한 수열

BOJ 1351

풀러 가기

힌트 0 (시간/공간 복잡도)

시간 복잡도는 \( O(\log^{2} N \log \log^{2} N) \)입니다.
공간 복잡도는 \( O(\log^{2} N) \)입니다.

힌트 1

점화식은 주어져 있습니다.
이제 값을 저장해야 하는데... 가능한 값이 너무 많습니다.
이를 줄일 수 있을까요?

힌트 2

0부터 \( N \)까지의 모든 답을 저장할 필요가 있을까요?
계산할 값들만 따로 계산할 수 있을까요?

힌트 3

\( A, B, C \)가 자연수일 때,
\( \left\lfloor \frac{ \left\lfloor \frac{A}{B} \right\rfloor }{C} \right\rfloor = \left\lfloor \frac{A}{BC} \right\rfloor \)를 만족합니다.

풀이

문제에 주어진 대로 재귀 형식의 DP를 구현해주시면 됩니다.
이 때 메모이제이션을 Map이나 Dict 등의 연관 배열로 해주시면 필요한 값들에 대해서만 메모이제이션해줄 수 있습니다.

시간 복잡도 계산은... 여기에 쓰진 않겠습니다. 힌트 3에서 유도 가능합니다.

코드 (C++)

#include <bits/stdc++.h>
#define FastIO ios_base::sync_with_stdio(0); cin.tie(0); cout.tie(0);
using namespace std;

using ll = long long;
map<ll, ll> memo;
ll p, q;

ll dp(ll x){
    if (memo.count(x) != 0){ return memo[x]; } // 메모이제이션
    if (x == 0){ return memo[x] = 1; } // 종료 조건
    return memo[x] = dp(x / p) + dp(x / q); // 점화식
}

void solve(){
    ll n; cin >> n;
    cin >> p >> q;
    cout << dp(n);
}

int main(){
    FastIO;
    solve();
}
