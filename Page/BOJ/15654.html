풀러 가기

힌트 0 (시간/공간 복잡도)

시간 복잡도는 \( O(2^N) \)입니다.
공간 복잡도는 \( O(N) \)입니다.

힌트 1

\( N \)이 많이 작습니다.
그냥 모든 경우를 둘러봐도 된다는 소리죠.

풀이

우선 입력으로 받은 수열을 정렬한 뒤,
만들 수 있는 길이 \( M \)의 모든 수열을 백트래킹으로 찾아주시면 됩니다.

코드 (C++)

#include <bits/stdc++.h>
#define endl '\n'
using ll = long long;
using namespace std;

int n, m;
int arr[10]; // 수열
bool chk[10]; // 선택 여부
vector<int> v;

void btk(int cnt){ // 현재 위치, 선택한 개수
    if (cnt == m){ // 정확히 M개를 선택했다면 출력 후 종료
        for (int x : v){ cout << x << ' '; }
        cout << endl;
        return;
    }
    for (int i = 1; i <= n; i++){
        if (!chk[i]){ // 선택하지 않은 수 중 아무거나 하나 선택
            chk[i] = 1; v.push_back(arr[i]);
            btk(cnt+1);
            chk[i] = 0; v.pop_back();
        }
    }
}

void solve(){
    cin >> n >> m;
    for (int i = 1; i <= n; i++){ cin >> arr[i]; }
    sort(arr+1, arr+n+1); // 수열 정렬
    btk(0);
}

int main(){
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    solve();
}
