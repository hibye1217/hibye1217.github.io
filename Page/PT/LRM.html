2. 변수, 함수 이름 재정의하기.

변수나 함수 이름이 _, __라면 알아보기 힘들겠죠.
그래서 각각의 함수에 적당한 이름을 부여했습니다.
하는 김에 using namespace std;도 추가했습니다.
#include <iostream>
using namespace std;

int f(int i, int n);
int g(int i, int n);
int h(int x);
int k(int x, int y);

int main(){
    int n;
    cin >> n;
    f(n, 9);
    return n^n;
}

int f(int i, int n){
    if (!n) return 0;
    f(i, h(n));
    cout << i << " * " << n << " = " << g(i, n) << endl;
    return i^i;
}

int g(int i, int n){
    return !(n - 1) ? i : i + g(i, h(n));
}

int h(int x){
    int y = -(~0);
    return k(x, y);
}

int k(int x, int y){
    if (x&y) return x^y;
    return k(x^y, y << -(~0));
}

x, y, f, g도 가독성이 그리 좋진 않지만, 적어도 _, __보단 낫죠.

5. 코드를 따라가며 복구하기.

우선 프로그램이 시작하면 n을 입력받은 뒤 f(n, 9)를 호출합니다.
f(n, 9)는 f(n, h(9))를 호출하는데,
h(x)는 k(x, 1)을 호출하고,
k(x, 1)은 x AND 1이 0이 아니면 x XOR y를 리턴하지만,
아니면 k(x XOR y, 2*y)를 재귀호출합니다.
그러면 우선 k 함수부터 보죠.

우선 k함수의 y변수는 항상 2의 제곱 형태가 됩니다.
이를 x와 AND연산을 하면, x의 특정 비트가 켜져 있는지 확인할 수 있게 됩니다.
즉 이 재귀함수는 x의 맨 끝 비트부터 검사하다가 켜진 비트에 도달하면 XOR한 결과를 리턴합니다. 아닌 경우는 XOR을 시행한 뒤 재귀호출을 하고요.
맨 뒤부터 0을 1로 채워나가다가 1을 만났을 때 이를 0으로 바꾸고 종료하는 것은 어떤 수에 -1을 하는 것과 같습니다.
고로 k(n, 1)은 1을 빼는 함수고, 자연스럽게 h(n)는 1을 빼는 코드라는 것을 알게 됩니다.
#include <iostream>
using namespace std;

int f(int i, int n);
int g(int i, int n);

int main(){
    int n;
    cin >> n;
    f(n, 9);
    return 0;
}

int f(int i, int n){
    if (n == 0) return 0;
    f(i, n-1);
    cout << i << " * " << n << " = " << g(i, n) << endl;
    return 0;
}

int g(int i, int n){
    if (n == 1) return i;
    else return i + g(i, n-1);
}

이제, f속에 있는 g의 정체를 알아내봅시다.
g(i, n)은 n이 1이면 i를 리턴하고,
아니면 g(i, n-1) + i를 재귀호출합니다.
한 번 호출할 때마다 n의 값이 1씩 줄어들고, n이 1이면 종료하므로
이 함수는 총 n회 호출됩니다.
각각에 호출에 대해서 g 함수는 이전 함수의 값에 i를 더하고, 결과값을 리턴합니다.
즉, 간단히 말하면 g(i, n)은 i를 n번 더하는 코드이고,
이는 g(i, n) = i*n이라는 걸 의미합니다.
#include <iostream>
using namespace std;

int f(int i, int n);

int main(){
    int n;
    cin >> n;
    f(n, 9);
    return 0;
}

int f(int i, int n){
    if (n == 0) return 0;
    f(i, n-1);
    cout << i << " * " << n << " = " << i*n << endl;
    return 0;
}

마지막으로, f함수입니다.
f(i, n)은 n = 0이면 종료하고,
아니면 f(i, n-1)을 호출합니다.
한 번 호출할 때마다 n의 값이 1씩 줄어들고, n이 0이면 종료하므로
호출 자체는 n+1회 하지만, n = 0이면 아무 일도 하지 않고 종료하기 때문에
사실상 n번 호출한다고 볼 수 있습니다. 또한 각각의 호출에서 i*n을 출력합니다.
호출을 먼저 한 다음에 출력을 할 때는 앞의 호출이 다 끝나야 출력을 할 수 있는 것이기 때문에
이 함수의 출력 순서는 1부터 시작해서, 원래 n값이었던 9까지 출력한 뒤 종료합니다.
즉, 이 함수는 1부터 9까지의 n에 대하여 i*n을 출력하고 있습니다.
#include <iostream>
using namespace std;

int main(){
    int n;
    cin >> n;
    for (int i = 1; i <= 9; i++) cout << n << " * " << i << " = " << i*n << endl;
    return 0;
}
