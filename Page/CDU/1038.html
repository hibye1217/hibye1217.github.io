CodeUp 1038

풀러 가기

힌트 0 (시간/공간 복잡도)

시간 복잡도는 \( O(1) \)입니다.
공간 복잡도는 \( O(1) \)입니다.

힌트 1

문제에서도 그림과 글로 주의를 주고 있습니다.
계산 결과가 int 범위를 넘어선다면,
어떤 자료형을 써야 할까요?

풀이

long long int 등의 자료형은 더욱 큰 수를 저장할 수 있으므로
long long int를 사용하여 입력 / 출력을 해주시면 됩니다.
long long int를 입/출력 할 때는 %lld를 사용하면 됩니다.

코드 (C)

#include <stdio.h>

int main(){
    long long int a, b;
    scanf("%lld %lld", &a, &b);
    printf("%lld", a+b);
}
