CodeForces 1A

Theatre Square

풀러 가기

번역

Berland의 중심 도시에 있는 Theatre Square는 크기가 \( N \times M \)인 직사각형 모양을 가지고 있습니다.
도시의 X주년 축제 중, Theatre Square를 정사각형 모양의 화강암 판석으로 채우기로 했습니다.
각 판석의 크기는 \( A \times A \)입니다.

Theatre Square를 채우기 위해 필요한 최소 개수의 판석은 몇 개일까요?
Theatre Square의 바깥 부분을 채워도 되지만, Theatre Square는 전부 채워져야 합니다.
판석을 부술 수 없으며, 판석의 모든 변들은 Theatre Square의 변과 평행해야 합니다.

힌트 0 (시간/공간 복잡도)

시간 복잡도는 \( O(1) \)입니다.
공간 복잡도는 \( O(1) \)입니다.

힌트 1

1차원적인 문제를 생각해보죠.
길이가 \( N \)인 선은 길이가 \( A \)인 선 몇 개로 채울 수 있을까요?

풀이

길이가 \( N \)인 선은 길이가 \( A \)인 선 \( \left\lceil \frac{N}{A} \right\rceil \)개로 채울 수 있습니다.
즉, Theatre Square의 세로를 채우기 위해서는 \( \left\lceil \frac{N}{A} \right\rceil \)개가 필요하고,
가로를 채우기 위해서는 \( \left\lceil \frac{M}{A} \right\rceil \)개가 필요합니다.
세로와 가로를 둘 다 채우기 위해서는 \( \left\lceil \frac{N}{A} \right\rceil \times \left\lceil \frac{M}{A} \right\rceil \)개가 필요하겠죠.

코드 (C++)

#include <bits/stdc++.h>
#define FastIO ios_base::sync_with_stdio(0); cin.tie(0); cout.tie(0);
#define endl '\n'
using ll = long long;
using namespace std;

void solve(){
    ll n, m, a;
    cin >> n >> m >> a;
    cout << ((n+a-1) / a) * ((m+a-1) / a); // ceil(N/A) * ceil(M/A)
}

int main(){
    FastIO;
    solve();
}
