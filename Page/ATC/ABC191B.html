풀러 가기

번역

정수로 이루어진 길이 \( N \)의 수열 \( A \)와, 정수 \( X \)가 주어집니다.
수열 \( A \)에서 \( X \)와 동일한 모든 원소를 제거한 뒤, 남은 원소들을 순서를 바꾸지 않고 합친 수열을 \( A' \)이라고 합시다.
\( A' \)을 출력하세요.

힌트 0 (시간/공간 복잡도)

시간 복잡도는 \( O(N) \)입니다.
공간 복잡도는 \( O(N) \)입니다.

힌트 1

수열 \( A \)에서 특정 값 \( X \)를 어떻게 제거할까요?

풀이

수열을 입력받은 뒤, 수열의 원소들을 돌면서 값이 \( X \)와 동일한지 비교한 뒤,
만약 \( X \)와 다르다면, 출력해주면 됩니다.

코드 (C++)

#include <bits/stdc++.h>
#define FastIO ios_base::sync_with_stdio(0); cin.tie(0); cout.tie(0);
using namespace std;

int a[100020];

int main(){
    FastIO;
    int n, x;
    cin >> n >> x;
    for (int i = 1; i <= n; i++){
        cin >> a[i];
    }
    for (int i = 1; i <= n; i++){
        if (a[i] != x){
            cout << a[i] << ' ';
        }
    }
}
