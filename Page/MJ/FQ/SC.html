Sqrt and Cbrt

질문 :

\( \sqrt[3]{ \dfrac{n}{2} } \) 와 \( \sqrt{ \dfrac{n}{3} } \) 이 동시에 자연수가 되게 만드는 가장 작은 자연수 \( n \)은?

답변 :

1. 일단 루트 안부터 자연수로.

우선 \( \cfrac{n}{2} \)이 자연수가 되려면 \( n \)은 \( 2 \)의 배수여야 합니다.
또한 \( \cfrac{n}{3} \)이 자연수가 되려면 \( n \)은 \( 3 \)의 배수여야 합니다.

2. \( n \)을 정의하기.

\( n \)은 \( 2 \)의 배수이면서 \( 3 \)의 배수여야 하고, 동시에 최소값이어야 하므로
\( n = 2^{a}3^{b} \) 로 정의할 수 있습니다.

3. 정의한 \( n \)을 대입하기.

\( \sqrt[3]{ \dfrac{n}{2} } = \sqrt[3]{ \dfrac{2^{a}3^{b}}{2} } = \sqrt[3]{ 2^{a-1}3^{b} } \)이고,
이 식이 자연수가 되려면 \( a-1 \text{ mod } 3 = 0 \)과 \( b \text{ mod } 3 = 0 \)이어야 합니다.
또한, \( \sqrt{ \dfrac{n}{3} } = \sqrt{ \dfrac{2^{a}3^{b}}{3} } = \sqrt{ 2^{a}3^{b-1} } \)이고,
이 식이 자연수가 되려면 \( a \text{ mod } 2 = 0 \)과 \( b-1 \text{ mod } 2 = 0 \)이어야 합니다.
얻은 4개의 식을 정리해서 써보면
\( a \text{ mod } 2 = 0 \), \( a \text{ mod } 3 = 1 \), \( b \text{ mod } 2 = 1 \), \( b \text{ mod } 3 = 0 \)
가 나옵니다.

4. 조건을 만족하는 최소값 \( a \), \( b \) 값 구하기.

\( a \text{ mod } 2 = 0 \) 에서 \( a \)는 \( 2 \)의 배수임을 알 수 있고,
\( a \text{ mod } 3 = 1 \) 에서 \( a \)는 \( 3x + 1 \)로 나타낼 수 있음을 알 수 있습니다.
\( 3x + 1 \)이 \( 2 \)의 배수가 되는 가장 작은 자연수 \( x \)는 \( 1 \)이고, 이 때의 \( a \) 값은 \( 4 \)입니다.
\( b \text{ mod } 3 = 0 \) 에서 \( b \)는 \( 3 \)의 배수임을 알 수 있고,
\( b \text{ mod } 2 = 1 \) 에서 \( b \)는 \( 2x + 1 \)로 나타낼 수 있음을 알 수 있습니다.
\( 2x + 1 \)이 \( 3 \)의 배수가 되는 가장 작은 자연수 \( x \)는 \( 1 \)이고, 이 때의 \( b \) 값은 \( 3 \)입니다.

5. \( n \) 값 구하기.

이제 마지막으로 \( n \)의 값을 구하면 됩니다.
\( n = 2^{a}3^{b} \)이고, \( a \)와 \( b \)의 최소값은 각각 \( 4 \)와 \( 3 \)이므로
\( n = 2^{a}3^{b} = 2^{4}3^{3} = 16 \times 27 = 432 \)가 나옵니다.
고로 \( n \)의 최소값은 \( 432 \)가 됩니다.


\( \mathbf{END.} \)
